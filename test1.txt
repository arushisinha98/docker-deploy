Open AWS Management Console
Navigate to EC2 console (instances panel)
Right-click docker instance and connect
Select EC2 Instance Connect (browser-based SSH connection)
Enter ec2-user as username
Connect

Add a custom tage to an image:
$ docker tag [IMAGE]:[ORIGINAL_TAG] [IMAGE]:[CUSTOM_TAG]
e.g. $ docker tag alpine:3.12.0 alpine:challenge

Create an image using a Dockerfile:
$ touch Dockerfile
$ vi Dockerfile
# FROM busybox:1.31.1
# MAINTAINER My Name = "ec2-user"
# EXPOSE 5000
<esc> :w :q

$ docker build ./ ## needs to be in directory with Dockerfile
$ docker run -lv -p 5000:5000 [IMAGE_ID]
$ docker tag [IMAGE_ID] myapp:0.0.1
